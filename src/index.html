Humphrey's BLOC Site

This is the homepage for Humphrey's BLOC Site. Here you will find a variety of utilities, tools and an API for the game BLOC.

Tools

  • Nations: A searchable, filterable list of all active (non vacation mode) nations in BLOC, updated daily
  • Nation Viewer: A viewer to show all the information on a single nation in BLOC, loads much faster than official BLOC pages
  • Planner: A planner, for planning out your game in BLOC
  • Target Finder: A taget finder for BLOC, will help you to find valid targets and quickly evaluate them without opening hundreds of bloc tabs

API

  • Scraper: A scraper for getting information from BLOC pages and storing it in a MongoDB database
  • API: An API for accessing the data from my scraper, used by all my tools
